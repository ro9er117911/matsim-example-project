# MATSim Project - Agent Work Logs

此資料夾包含專案的結構化知識庫，按不同專業領域的 Agent 組織。

## 專案概述

**專案名稱**: MATSim Example Project
**專案類型**: 多代理人運輸模擬 (Multi-Agent Transport Simulation)
**主要語言**: Java 21
**建置工具**: Maven 3.9.8
**MATSim 版本**: 2025.0
**座標系統**: EPSG:3826 (TWD97 / TM2 zone 121, Taiwan)

## Agent 專業領域劃分

### 1. project-structure-specialist
**責任範圍**: 專案架構、模組組織、建置系統、依賴管理

**工作日誌**:
- `project-overview.md` - 專案整體架構概覽
- `architecture-analysis.md` - 詳細架構分析
- `build-system-analysis.md` - Maven 建置系統與依賴管理
- `module-inventory.md` - 模組清單與職責

### 2. dev-master
**責任範圍**: 程式碼實作、開發指南、編碼模式

**工作日誌**:
- `development-guide.md` - 開發者指南
- `entry-points-guide.md` - 應用程式入口點說明
- `pt-tools-implementation.md` - PT 工具實作細節

### 3. test-assistant
**責任範圍**: 測試框架、測試案例、品質保證

**工作日誌**:
- `testing-infrastructure.md` - 測試基礎設施分析
- `test-coverage-analysis.md` - 測試覆蓋率分析
- `testing-best-practices.md` - 測試最佳實踐

### 4. product-manager
**責任範圍**: 功能規格、需求文檔、使用者文檔

**工作日誌**:
- `feature-catalog.md` - 功能目錄
- `pt-toolchain-specification.md` - PT 工具鏈規格
- `scenario-documentation.md` - 場景與配置文檔

### 5. research-mve-designer
**責任範圍**: 研究實驗、參數調優、最小可行實驗

**工作日誌**:
- `pt-mapping-research.md` - PT 對應研究與調優
- `configuration-tuning-guide.md` - 配置參數調優指南
- `experimental-workflows.md` - 實驗性工作流程

## 使用指南

每個 Agent 在開始工作前應：
1. 閱讀此 README 了解專案概況
2. 查看自己專業領域的工作日誌
3. 根據需要更新或補充工作日誌
4. 記錄重要決策和發現
